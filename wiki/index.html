Projet: DAW

Ce dossier (wiki) n'a pas à être partagé.
Il est uniquement disponible pour les développeurs afin de faciliter la compréhension de l'architecture mise en place.

Il ne respecte pas la consigne qui dit qu'on doit faire tout nous même, car il utilise boostrap. Mais comme il n'a pas vocation a être rendu et n'est pas inclus dans le site...


Mise en place du serveur

Afin de mettre en place le serveur vous aurez besoin de :

  • MAMP
  • Un clone du github

Les étapes à suivre sont:

  1. Installer MAMP
  2. Déplacer l'intégralité du git dans le dossier htdocs de MAMP.
  3. Configurer MAMP :
    • Utiliser PHP 8 au minimum
    • Utiliser Apache
    • Définir le fichier root (ou working directory) sur le dossier public
  4. Se rendre sur la page de phpmyadmin= lien avec le port 8888 :
    1. Créer une base de données nommée daw: à gauche de l'écran
    2. Se rendre dans la base de données, puis dans l'onglet SQL entrer:
      SET NAMES utf8mb4;
      SET FOREIGN_KEY_CHECKS = 0;
      DROP TABLE IF EXISTS `account`;
      CREATE TABLE `account` (
      `id` int(11) NOT NULL AUTO_INCREMENT,
      `login` varchar(255) NOT NULL,
      `password` varchar(255) NOT NULL,
      `email` varchar(255) DEFAULT NULL,
      `create_date` datetime DEFAULT NULL,
      `tel` varchar(255) DEFAULT NULL,
      `born` datetime DEFAULT NULL,
      PRIMARY KEY (`id`)
      ) ENGINE=InnoDB AUTO_INCREMENT=4 DEFAULT CHARSET=utf8;
      BEGIN;
      INSERT INTO `account` VALUES (1, 'taku', 'test', 'cc@nico.fr', '2022-03-10 12:03:17', '0633596086', '2001-06-08 12:00:00');
      INSERT INTO `account` VALUES (2, 'root', 'test2', 'cc@nico.fr', '2022-03-10 12:03:31', '0633596086', '2001-06-08 12:00:00');
      INSERT INTO `account` VALUES (3, 'other1', 'test22', 'cEc@nico.fr', '2022-03-10 12:03:38', '0633596086', '2001-06-08 12:00:00');
      COMMIT;
      SET FOREIGN_KEY_CHECKS = 1;
  5. Le site est désormais fonctionnel.
