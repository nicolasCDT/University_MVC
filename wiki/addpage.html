Projet: Ajouter une page

Pour ajouter une page, il faut commencer par choisir le contrôleur dans lequel on va l'ajouter.
Une fois ceci fait, il faut ajouter le lien et le nom de la méthode dans le constructeur.

Par exemple, nous allons ajouter une page qui dit Coucou tout le monde, accessible au lieu mon-site.com/hello/world/

On ajoute dans le constructeur : $this->routes["/hello/world:"] = "maFonction";
On ajoute ensuite une fonction réponse dans le même contrôleur:
  public function index(Request $request): Response {
    return new Response('Coucou tout le monde');
  }
