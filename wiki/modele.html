Projet: Modèle

Les modèles sont découpés en deux grosses parties:

  • Entités
  • Dépôts

Les entités

Les entités modélise une ligne d'une table de la base de données. Dans le gros : C'est la traduction d'une ligne SQL en objet PHP.
Les entités peuvent être générées grâce au script make_entity.sh (Shell).
Attention: si vous générez une entité, il ne faut pas oublier d'exécuter le fichier .sql généré également !

A partir d'un contrôleur, vous pouvez créer une entité en l'instanciant. Il ne faut pas renseigner l'identifiant, la base de données s'en charge.
Pour l'ajouter à la base de données, il faut utiliser la méthode add: $this->getDBManager->add(entity);

Pour récupérer une entité déjà en base de données, il faut passer par les dépôts.

Les dépôts

Les dépôts modélisent quant à eux une table entière. C'est une liste organisée d'entités.
Par exemple, pour le dépôt AccountRepository, on peut récupére l'ensemble des comptes en base de données en faisant $this->getRepository(AccountRepository::class)->findAll().
Les méthodes principales sont :

  • findAll() : Renvoie toutes les entités
  • findBy() : Trouve les entités grâce à une liste de condition (tableau 2D propriété => valeur)
  • findOneBy() : Trouve la première entité qui respecte les conditions
